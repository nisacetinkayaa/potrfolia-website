# potrfolio-website

Bu proje, kişisel bir portföy sitesidir. HTML ve CSS kullanılarak hazırlanmıştır.
JavaScript veya herhangi bir framework kullanılmamıştır.

Özellikler

Tamamen statik yapı (yalnızca HTML ve CSS)

Hakkımda, Projeler, ve İletişim bölümleri

Minimal ve sade arayüz

Kolayca özelleştirilebilir yapı

Klasöre gidin:

cd portfolio-site


index.html dosyasını tarayıcınızda açın:

open index.html


veya dosyayı sürükleyip tarayıcıya bırakın.

Özelleştirme

style.css dosyasını düzenleyerek renk paletini, yazı tiplerini ve düzeni değiştirebilirsiniz.

index.html içindeki metinleri kendi bilgilerinizle değiştirin.

Kendi proje bağlantılarınızı ve sosyal medya ikonlarınızı ekleyin.

 Lisans

Bu proje açık kaynaklıdır. İstediğiniz gibi kullanabilir, düzenleyebilir ve paylaşabilirsiniz.
